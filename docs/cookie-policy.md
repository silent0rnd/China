# Cookie Policy

## 1. Статус

Это draft для юридической проверки. Production-домен, counter ID и окончательные сроки хранения пока не настроены.

## 2. Модель

Сайт использует консервативный режим:

- необходимые функции доступны сразу;
- необязательная аналитика выключена по умолчанию;
- Яндекс.Метрика загружается только после согласия;
- отказ не блокирует контент, формы, телефоны и мессенджеры;
- настройки можно изменить через footer.

## 3. Категории

### Необходимые

Нужны для:

- сохранения выбранных cookie-настроек;
- защиты формы от повторной отправки;
- временной server session при работе формы.

Проектные имена:

| Имя | Назначение | Срок |
|---|---|---|
| `gorizont_cookie_consent` | категории, version и timestamp | 180 дней, draft |
| server session cookie | submission token и idempotency | до 30 минут или закрытия session |

Точное имя server session определяется реализацией. В cookie нельзя помещать содержимое формы.

### Analytics

- сервис: Яндекс.Метрика;
- counter ID: not configured;
- загрузка: после consent;
- cookies и сроки: определяются фактической конфигурацией сервиса и должны быть проверены перед публикацией.

## 4. Banner

Кнопки:

- «Принять все»;
- «Отклонить необязательные»;
- «Настроить».

В настройках:

- необходимые, всегда включены;
- analytics, выключены по умолчанию до выбора пользователя.

## 5. Сохранение выбора

Сохраняются:

- policy version;
- выбранные категории;
- timestamp.

Изменение решения доступно через ссылку «Настройки cookie» в footer.

## 6. Аналитические события

Без analytics consent цели не отправляются. После согласия отправляются только обезличенные события из `analytics-plan.md`.

Запрещено передавать в аналитику:

- имя;
- телефон или мессенджер;
- email;
- адреса;
- комментарии;
- свободное описание груза;
- имя и содержимое файла.

## 7. Сторонние ссылки

Переход в WhatsApp, Telegram или WeChat является самостоятельным действием пользователя. Эти сервисы могут применять собственное хранение после перехода.

## 8. Открытые поля

- production-домен;
- counter ID;
- Webvisor on/off;
- точные cookies Яндекс.Метрики;
- окончательный срок consent cookie;
- юридическая редакция текста.
